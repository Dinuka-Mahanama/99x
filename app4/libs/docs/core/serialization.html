Serialization#

Most of objects, such as BarcodeCapture provide methods to convert to and from JSON strings. The main use of this serialization is to be able to capture the current configuration of the object and restore it at a later point on any device. This can be especially useful to debug issues without needing to filter through the code base of an entire project.

Creating from JSON#

There are multiple levels at which you can create objects from JSON. The following sample shows how to construct a BarcodeCapture instance from a JSON string, it will deserialize the barcode settings plus additional properties set on the barcode capture mode. Of course it is also possible to just create the BarcodeCaptureSettings from JSON without the BarcodeCapture instance around it.

You can either use the standard JSON deserialization.

DataCaptureContext dataCaptureContext = ...;
// The settings represented in a json string.
String jsonString = ...;
try {
  // Create the settings from it.
  BarcodeCapture BarcodeCapture.fromJson(dataCaptureContext, jsonString);
  // Use the barcode capture mode.
} catch (Exception e) {
  // The json failed to be parsed.
}

Updating from JSON#

Just like creating new objects, it is possible to update already existing objects from JSON. The following sample shows how to update a BarcodeCapture instance from a JSON string.

You can either use the standard JSON deserialization.

// The settings represented in a json string.
String jsonString = ...;
try {
  // Create the settings from it.
  BarcodeCapture BarcodeCapture.updateFromJson(jsonString);
  // Use the updated barcode capture mode.
} catch (Exception e) {
  // The json failed to be parsed.
}
